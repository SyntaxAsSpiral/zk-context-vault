# Planetary Hours and Tattwas

## Planetary Hours

Planetary hours divide daylight into 12 variable-length hours and nighttime into 12 variable-length hours. The ruler sequence follows the Chaldean order:

```text
Saturn → Jupiter → Mars → Sun → Venus → Mercury → Moon → repeat
```

The first daylight hour is ruled by the planetary ruler of the weekday:

| Weekday | Day Ruler |
|---------|-----------|
| Sunday | Sun |
| Monday | Moon |
| Tuesday | Mars |
| Wednesday | Mercury |
| Thursday | Jupiter |
| Friday | Venus |
| Saturday | Saturn |

Interpret planetary hours as qualitative emphasis, not deterministic command.

## Tattwas

The bundled script uses five tattwas in fixed 24-minute segments from the most recent sunrise:

```text
Prithvi → Apas → Tejas → Vayu → Akasha → repeat
```

Day-specific starting element used by the bundled script:

| Weekday | Starting Tattwa |
|---------|-----------------|
| Sunday | Tejas |
| Monday | Apas |
| Tuesday | Tejas |
| Wednesday | Prithvi |
| Thursday | Akasha |
| Friday | Apas |
| Saturday | Vayu |

## Interpretive Keywords

| Signal | Keywords |
|--------|----------|
| Saturn | structure, boundary, discipline, time |
| Jupiter | expansion, teaching, generosity, meaning |
| Mars | action, conflict, courage, cutting |
| Sun | clarity, leadership, vitality, visibility |
| Venus | harmony, beauty, relation, ease |
| Mercury | language, code, trade, analysis |
| Moon | memory, care, intuition, fluctuation |
| Prithvi | earth, stability, durability |
| Apas | water, flow, emotion, restoration |
| Tejas | fire, transformation, heat |
| Vayu | air, movement, thought |
| Akasha | space, potential, subtlety |
