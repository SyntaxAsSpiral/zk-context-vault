# Semantic-JSON Workflows Skill

*Master the canvas-to-structured-data anticompiler for visual cognitive modeling.*

## What This Skill Provides

This skill teaches the complete Semantic-JSON workflow - from visual design in Obsidian Canvas through semantic compilation to structured data export. It covers the anticompiler philosophy, spatial semantic analysis, and advanced integration patterns.

**Core capability**: Transform visual models into deterministic, human-readable structured data while preserving spatial semantics.

## Skill Components

### [Main Skill Document](SKILL.md)
Comprehensive guide covering:
- **Anticompiler Philosophy**: Understanding visual-to-structured-data transformation
- **Spatial Semantic Compilation**: Position, containment, color, and flow topology
- **Canvas Design Patterns**: Node structures, edge semantics, and color taxonomy
- **Compilation Workflows**: Basic compilation, flow topology, and pure JSON export
- **Advanced Patterns**: Edge embedding, content extraction, and import scaffolding
- **Integration**: Workshop recipes, Kiro specs, and Templater workflows
- **Quality Assurance**: Validation, debugging, and performance optimization

## Key Learning Outcomes

After mastering this skill, you'll be able to:

1. **Design semantic canvases** with meaningful spatial organization and visual encoding
2. **Compile visual models** into stable, deterministic JSON structures
3. **Process canvas data** programmatically for integration with other systems
4. **Validate and debug** canvas structures and compilation results
5. **Integrate workflows** with workshop recipes, Kiro specs, and templater systems
6. **Optimize performance** for large-scale canvas processing
